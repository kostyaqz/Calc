1) Положительные тесты:
	а) Сложение, вычитание, деление, умножение целых положительных чисел:
		Запустить консоль;
		Ввести первое положительное число;
		Ввести каждый из операторов для всех 4х итераций (+,-,/,*);
		Ввести второе положительное число;
		Получить ответ;
		Закончить работу с приложением.
	б) Сложение, вычитание, деление, умножение целых отрицательных чисел:
		Запустить консоль;
		Ввести первое отрицательное число;
		Ввести каждый из операторов для всех 4х итераций (+,-,/,*);
		Ввести второе отрицательное число;
		Получить ответ;
		Закончить работу с приложением.
	в) Сложение, вычитание, деление, умножение целых отрицательных и положительных чисел (первое и второе число с разными знаками):
		Запустить консоль;
		Ввести первое отрицательное/положительное число;
		Ввести каждый из операторов для всех 4х итераций (+,-,/,*);
		Ввести второе отрицательное/положительное число;
		Получить ответ;
		Закончить работу с приложением.
	г) Сложение, вычитание, деление, умножение дробных чисел переменного знака:
		Запустить консоль;
		Ввести первое дробное число;
		Ввести каждый из операторов для всех 4х итераций (+,-,/,*);
		Ввести второе дробное число;
		Получить ответ;
		Закончить работу с приложением.
	д) Деление на ноль:
		Запустить консоль;
		Ввести первое число;
		Ввести оператор деления;
		Ввести ноль;
		В качестве ответа получить "Отношение введенных чисел равно ?";
		Закончить работу с приложением;
	е) Продолжение работы с калькулятором:
		Запустить консоль;
		Ввести первое число;
		Ввести любой из операторов;
		Ввести второе число;
		Получить ответ;
		На предложение продолжить работу с калькулятором ввести "д";
		В результате удостовериться, что программа вернула нас на этап ввода первого числа.
	е) Завершение работы с калькулятором:
		Запустить консоль;
		Ввести первое число;
		Ввести любой из операторов;
		Ввести второе число;
		Получить ответ;
		На предложение продолжить работу с калькулятором ввести любой символ кроме "д";
		В результате удостовериться, что программа закрыла консоль.


2) Отрицательные тесты:
	а) Ввод любого другого символа вместо чисел:
		Запустить консоль;
		Ввести любой символ (исключая цифры);
		Получить ошибку при работе с программой в среде визуальной студии;
		Закончить работу с приложением.
	б) Ввод любого другого символа вместо предлагаемых операторов:
		Запустить консоль;
		Ввести первое число;
		Ввести любой символ кроме операторов;
		Ввести второе число;
		Получить ошибку "Неверный оператор";
		Закончить работу с приложением.
	в) Ввод числа вместе с символом:
		Запустить консоль;
		Ввести любое число, добавив к нему в конце любой символ (возможны ошибки при вводе пользователя, например "1+";
		Получить ошибку при работе с программой в среде визуальной студии;
		Закончить работу с приложением.
	г) Ввод дробного числа через точку:
		Запустить консоль;
		Ввести любое дробное число через точку;
		Получить ошибку при работе с программой в среде визуальной студии;
		Закончить работу с приложением.
